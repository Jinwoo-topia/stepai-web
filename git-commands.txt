== Git 명령어 패턴 ==

기본 플로우:
git add .
git commit -m "feat: 새기능설명"  
git push

한 줄로:
git add . && git commit -m "메시지" && git push

커밋 메시지 패턴:
- feat: 새 기능
- fix: 버그 수정  
- refactor: 리팩토링
- style: 스타일 변경

== 프로젝트 설정 ==
- 좌우 여백: 320px
- 브레이크포인트: 헤더와 동일
- 카드 겹침: 20px 오프셋
- 자동 회전: 3초마다
- 카드 크기: 100%, 90%, 80%
- 하단 정렬
